# Project 02 - Subnetting Notes

## Why Subnetting?

Subnetting divides a large network into smaller networks.

Benefits:
- Reduces broadcast traffic
- Improves security
- Makes networks easier to manage
- Uses IP addresses more efficiently

## CIDR Notation

| CIDR | Subnet Mask |
|------|-------------|
| /24 | 255.255.255.0 |
| /25 | 255.255.255.128 |
| /26 | 255.255.255.192 |
| /27 | 255.255.255.224 |

## Key Takeaways

- IPv4 addresses contain 32 bits.
- The CIDR prefix indicates how many bits are used for the network.
- The remaining bits are available for hosts.
